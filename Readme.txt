Chatbox Ansiedade: Seu Companheiro Digital para o Bem-Estar Emocional 

[chatbox_ansiedade](https://github.com/silvafarol2/chatbot_ansiedade)

**Em um mundo onde a ansiedade pode, por vezes, obscurecer a clareza e a paz interior, o "Alívio e Informação" surge como um farol de esperança e um guia prático para o seu bem-estar emocional. Este não é apenas um chatbot; é um amigo digital dedicado a fornecer conhecimento, técnicas de relaxamento ao seu alcance e a direção necessária para encontrar apoio profissional em sua comunidade.**

### ✨ Uma Jornada Rumo à Tranquilidade ao Seu Alcance

O "Alívio e Informação" foi meticulosamente desenvolvido em Python, impulsionado pela inteligência do modelo Gemini, para criar uma experiência de interação fluida, informativa e, acima de tudo, acolhedora. Nosso objetivo é desmistificar a ansiedade, oferecer ferramentas práticas para o autocuidado e conectar você com recursos de ajuda profissional de forma simples e eficaz.

### 🚀 Funcionalidades que Cuidam de Você:

  * **Desvendando a Ansiedade:** Obtenha informações claras e concisas sobre o que é a ansiedade, seus diferentes aspectos e como ela pode se manifestar. (Esta funcionalidade será continuamente aprimorada com mais detalhes e nuances.)
  * **Seu Kit de Relaxamento Digital:** Explore e aprenda técnicas de relaxamento comprovadas, como a respiração profunda e o relaxamento muscular progressivo, com explicações passo a passo para que você possa incorporá-las facilmente em sua rotina. Além disso, sinta-se à vontade para perguntar sobre outras técnicas que lhe interessem\!
  * **Encontre Apoio Perto de Você:** Com a nossa funcionalidade de busca por ajuda profissional, você pode identificar rapidamente hospitais (tanto do SUS quanto particulares) e clínicas de psicologia com atendimento de emergência em sua cidade e estado (*atualmente com foco em fornecer informações relevantes com base na sua localização*). Queremos facilitar o acesso ao suporte quando você mais precisa.

### 🛠️ Como Integrar o "Alívio e Informação" em Seu Dia a Dia:

1.  **Um Passo de Cada Vez:** Comece clonando este repositório para o seu computador:
    ```bash
    git clone https://github.com/silvafarol2/chatbot_ansiedade.git
   cd chatbot_ansiedade
    ```
    *(Substitua `https://github.com/silvafarol2/chatbot_ansiedade.git` pela URL real do seu repositório no GitHub)*

2.  Mergulhe no Nosso Universo:** Navegue até o diretório do projeto:
    ```bash
    cd chatbox_ansiedade
    ```
3.  Pré-requisitos Essenciais:** Certifique-se de ter o Python instalado em seu sistema para dar vida ao nosso chatbot.
4.  Instalando as Dependências:** Para executar o chatbot, você precisará das bibliotecas `google-genai` e `google-generativeai`. Se estiver usando o Google Colab, você pode instalá-las com o seguinte comando:
    ```bash
    !pip install google-genai google-generativeai
    ```
    Se estiver executando localmente, use o seguinte comando no seu terminal:
    ```bash
    pip install google-genai google-generativeai
    ```    

5.  **Conecte-se Conosco:** Execute o script principal do chatbot (geralmente chamado `chatbot_ansiedade.py`):
    ```bash
    python chatbot_ansiedade.py
    ```
6.  **Sua Jornada Começa Aqui:** Interaja com o chatbot diretamente no seu terminal. Digite comandos como "relaxar" para explorar as técnicas, "ajuda" para buscar suporte local ou simplesmente faça suas perguntas sobre ansiedade.

### 🧘‍♀️ Técnicas de Relaxamento ao Seu Dispor:

Atualmente, o "Alívio e Informação" pode guiá-lo através das seguintes práticas de relaxamento:

  * Respiração Profunda
  * Relaxamento Muscular Progressivo
  * Sinta-se à Vontade para Explorar: Não hesite em perguntar sobre outras técnicas que você gostaria de conhecer\!

### 📍 Encontrando o Caminho para o Apoio Profissional:

Quando você sentir a necessidade de buscar ajuda profissional, digite "ajuda". O "Alívio e Informação" solicitará sua cidade e estado para fornecer informações sobre hospitais e clínicas com atendimento de emergência em sua região. Você também pode especificar se sua preferência é por instituições que atendem pelo SUS ou particulares, para refinar ainda mais a sua busca.

Construído com Cuidado e Tecnologia:

* **Linguagem de Programação:** Python (a espinha dorsal da nossa inteligência).
* **Modelo de Linguagem:** Gemini (potencializando conversas informativas e engajadoras através da API do Google AI Studio).
* **Ambiente de Desenvolvimento (Opcional):** Google Colab (utilizado para conectar a API Gemini em Python).
* **Chave de API:** Google AI Studio API Key (necessária para autenticação e uso da API Gemini).
